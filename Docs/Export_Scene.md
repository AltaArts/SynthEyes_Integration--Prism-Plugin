# **Scene Export**

SynthEyes has many formats to export a solved scene.  The Prism integration handles several of the most popular formats using the SceneExport state in the State Manager.  Currently supported formats are .USD, .FBX, .ABC, Blender (.py), Maya (.ma), Blackmagic Fusion (.comp), and Nuke (.nk).
